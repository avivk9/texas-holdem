## Usage:
1) Compile all *.java files, using an IDE or command prompt
```
javac *.java
```
2) Run the server Main.class compiled run file, using an IDE or command prompt
```
java server.Main
```
3) Run as many clients as you want with the correct server ip
```
java client.Main
```
